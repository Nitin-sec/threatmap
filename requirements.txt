# ── Local SLM (recommended — no API key needed) ────────────────────────────
#
#   Provides AI triage using a locally-downloaded GGUF model.
#   Run once to install, then run setup_slm.py to download the model.
#
#   CPU-only (safe for all VMs):
#     pip install llama-cpp-python huggingface_hub
#
#   With NVIDIA GPU acceleration (optional, much faster):
#     pip install llama-cpp-python[cuda] huggingface_hub
#
#   With Apple Silicon GPU (optional):
#     pip install llama-cpp-python[metal] huggingface_hub
#
#   After installing, run:
#     python3 setup_slm.py              # downloads Qwen2.5-1.5B (~1 GB, default)
#     python3 setup_slm.py --phi3       # downloads Phi-3-mini (~2.2 GB, better)
